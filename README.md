# Personal Website Project

This repository contains instructions and structure for building a **full personal website** from scratch using HTML, CSS, and optional JavaScript. The website will serve as a professional portfolio and personal homepage for your projects, skills, and contact information.

All sensitive content (ITAR, proprietary data) should be **abstracted or non-dimensionalized**.

---

## 1. Website Overview

The personal website should include the following sections:

1. **Home Page**
   - Elevator pitch / short introduction
   - Areas of expertise
   - Links to projects, resume, and contact page

2. **About Me**
   - Your background, skills, and interests
   - Professional summary (keep it ITAR-safe)

3. **Projects**
   - List of projects
   - Each project has its own page or modal
   - Include representative images, plots, or descriptions
   - Avoid including controlled technical data

4. **Resume**
   - Upload a PDF
   - Provide a download link

5. **Contact**
   - Email, LinkedIn, GitHub
   - Optional contact form (sanitized)

6. **ITAR / Compliance Notice**
   - Clear statement that all public content is abstracted
